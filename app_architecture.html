Key Files

backtest_service.py Pipeline orchestration, LLM calls
LLM_FIX_PYTHON_PROMPT Prompt for fixing/filling placeholders

Unknown Block Handling

When a block isn't in JSON map:

  1. Log: WARNING: Unknown indicator 'XYZ'
  2. Add placeholder: # TODO: LLM_FILL
  3. LLM fills in the implementation
  4. Return warning in response

Example

# TODO: LLM_FILL - Unknown indicator 'XYZ' - params: {'period': 10}
# self.unknown_xyz_0 = self.I(???, self.data.Close, ???)
